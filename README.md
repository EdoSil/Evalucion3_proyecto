Proyecto Evaluación 3 – Aplicación Web con Flask

Este proyecto corresponde a la Evaluación 3 del taller de programación WEB, donde se solicita desarrollar una aplicación web utilizando Python, Flask, plantillas HTML con Jinja2, manejo de formularios y almacenamiento de datos en memoria.

Descripción General

La aplicación cuenta con un menú principal que permite acceder a dos ejercicios y a una página de memoria donde se almacenan los registros ingresados por el usuario.

La estructura es la siguiente:

Ejercicio 1 – Notas y Asistencia
Permite ingresar tres notas (entre 10 y 70) y un porcentaje de asistencia (0 a 100).
El sistema calcula:

Promedio de las notas

Determina si el estudiante está APROBADO o REPROBADO, con base en:

Promedio ≥ 40

Asistencia ≥ 75%

Ejercicio 2 – Longitud de Nombres
Permite ingresar tres nombres distintos.
El sistema muestra:

El nombre con mayor cantidad de caracteres

La cantidad de caracteres de ese nombre
Además, los registros quedan almacenados en memoria.

Memoria de Registros
Página donde se muestran todos los nombres ingresados en el Ejercicio 2, junto con el nombre más largo y su cantidad de caracteres.
